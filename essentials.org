* kpm

  - goal: $\sigma^{reg}(\omega) = \frac{1}{Z \omega} j(y + \omega, y)(e^{-\beta y} - e^{-\beta(y + \omega)})$, Z from DOS integral, but can be skipped if sigma is a.u.
  - $j(x,y) = \sum_{n, m} \mu_{nm} h_{nm} g_n g_m T_n(x), T_m(x) \cdot \frac{1}{\pi^2 \sqrt{(1-x^2)(1-y^2)}}$
  - $\mu_{nm} = \frac{1}{D} \text{Tr}[T_n(H')J T_m(H') J]$, where $H'$ is rescaled with evs between -1 and 1
  - $h_{nm} = \frac{2}{1 + \delta_{n,0}} \cdot \frac{2}{1 + \delta_{m,0}}$
  - $T_n(x) = \cos[n \arccos(x)], T_0(x) = 1, T_{-1}(x) = T_1(x) = x, T_{m+1}(x) = 2x T_m(x) - T_{m-1}(x)$
  - $g_n$ kernel damping (e.g. 1, but then gibbs ringing)
